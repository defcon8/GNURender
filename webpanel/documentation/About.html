About GNURender



GNURender is an advanced render-farm management application designed for Blender. It allows multiple users to browse to an easy-to-use web panel and submit their .blend files for rendering. GNURender divides the projects into separate frame tasks which are distributed to available rendering nodes on the network. All graphical- and resource data is stored as binary data (BLOB) inside a MySQL database, which is one of the key aspects of this software. At this moment GNURender is in ALPHA stage, which means the software is under heavy development. Also there is no user manual at this stage, we try to make the user interface functionality transparent and straightforward as possible.
