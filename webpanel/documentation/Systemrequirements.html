System requirements:


This install manual is based on a Debian Linux 7 (Wheezy) point of view, but can also be used as guidelines for Windows & Macintosh systems. GNURender software consists mainly out of 2 parts, the web panel and one or multiple render nodes. The web panel part is coded in PHP and uses a MySQL database for data storage. The node is coded in C# and compiled with MONO and therefore (theoretically) should run on Linux, Windows & OSX.


The whole package is developed on Debian Linux, but the nodes are currently tested

on both Debian & Windows 7.


Webpanel:


  • Apache2
  • PHP5
  • MySQL



From this point we assume that you already have a working LAMPP stack, if not there are plenty guides available on the internet to accomplish this. The web panel needs PHP extensions MySQL and MySQLi to be enabled. Keep in mind that each node needs a direct connection to the database server. Therefore MySQL needs to allow incoming connections from local host and PC's other then local host.


Node:


Packages needs to be installed on system:

  • MONO
  • unzip
  • ffmpeg
